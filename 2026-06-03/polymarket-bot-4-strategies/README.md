# Polymarket Bot — 4 Strategies

**Source:** https://github.com/MrFadiAi/Polymarket-bot

## What It Does

A Polymarket trading bot with **4 bundled strategies** and active development (v3.1 — January 2026). Includes:
- Enhanced Risk Management
- Smart Money improvements
- Dynamic position sizing
- Dashboard with Multi-strategy support
- Auto-rotation between strategies

## Why It Matters

The multi-strategy auto-rotation is a practical approach — rather than betting on one strategy, let the system switch between them based on market conditions. The active development (3 major versions in recent months) suggests a serious project.

## Risks
- Less technical documentation than some alternatives
- Strategy details are opaque

## Implementability: 3/5

Good for quick start with pre-built strategies. The auto-rotation feature is practical but strategy logic isn't transparent.
