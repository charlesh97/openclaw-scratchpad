# Summary: Designing AMMs for Combinatorial Securities

**Authors:** Multiple
**Published:** November 2024
**arXiv:** 2411.08972

## TL;DR
Formal framework for building AMMs that support combinatorial securities. Shows this can be done efficiently by reducing to range update problems. Important for next-gen prediction markets.

## Relevance to Vega Research
Forward-looking. Current Polymarket uses binary conditional tokens, but the trend is toward more complex instruments. Our arb detection should be designed extensibly to handle combinatorial securities when they arrive.
