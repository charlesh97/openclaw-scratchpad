# OctoBot Prediction Market

**Source:** https://github.com/Drakkar-Software/OctoBot-Prediction-Market  
**Recommendation:** MEDIUM

## What It Does

A free open-source Polymarket trading robot built on top of OctoBot (established crypto trading bot framework). Supports copy trading of Polymarket profiles and arbitrage strategies with a user-friendly interface, Telegram integration, and upcoming mobile app.

## Key Features

- Copy trading of Polymarket profiles
- Arbitrage strategy support
- User-friendly web + Telegram interface
- Risk-free paper trading mode
- Self-custody (keys stay on your device)
- Upcoming Kalshi integration

## Why It Matters

Backed by the established OctoBot project, which has been a successful crypto trading bot framework for years. The solid user interface and community support make it accessible. The paper trading feature is valuable for strategy testing.

## Implementability: 4/5

Well-documented, Python-based, runs on the established OctoBot framework. User-friendly interface means less custom development. Good for getting started quickly.

## Risks

- Copy trading still under development (WIP features)
- Less sophisticated than dedicated arbitrage bots
- Dependent on OctoBot ecosystem updates
