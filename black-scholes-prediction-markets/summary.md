# Summary: Toward Black Scholes for Prediction Markets

**Authors:** Shaw Dalen  
**Date:** October 2025 (v2: April 2026)  
**arXiv:** 2510.15205

Proposes a complete stochastic calculus framework for prediction markets. The logit jump-diffusion model treats probability as a martingale with jump components, enabling options-like risk factor quoting. If adopted by platforms like Polymarket, this would create a new derivative layer (variance swaps, correlation products, corridor instruments) for prediction markets.
