# Polymarket Architecture Reference

**Source:** https://github.com/loongxjin/polymarket-architecture

## What It Is
A comprehensive technical documentation repository covering Polymarket's entire architecture — the world's largest decentralized prediction market platform. Written as a developer's guide to understanding every layer of Polymarket.

## Coverage
- **System Architecture:** Hybrid decentralized design combining CLOB (off-chain) + CTF (on-chain ERC-1155)
- **Core Concepts:** Conditional tokens, YES/NO markets, Condition IDs
- **Trading Mechanics:** Order matching, split/merge operations, arbitrage detection
- **Smart Contracts:** CTF Core, Exchange contracts, direct contract interaction patterns
- **API Integration:** CLOB API, Data API, development examples
- **Advanced Topics:** Gnosis Safe integration, market settlement, UMA oracle, gas optimization
- Complete in both English and Chinese

## Why It Matters
Building arbitrage bots and trading strategies on Polymarket requires deep understanding of how the platform works under the hood. This is the single most comprehensive third-party reference — essential reading for anyone building on Polymarket.
