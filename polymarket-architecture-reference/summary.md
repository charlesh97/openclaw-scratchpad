# Summary: Polymarket Architecture Reference

A comprehensive technical reference covering Polymarket's full stack — from core CTF (Conditional Token Framework) protocol on Polygon to CLOB off-chain order matching, smart contract layer (CTF Core, Exchange), API integration (CLOB API, Data API), and advanced topics like Gnosis Safe integration and UMA oracle resolution. Written for developers building trading bots, arbitrage scanners, or market-making strategies on Polymarket.

**Key takeaway:** Understanding the split/merge mechanism for conditional tokens is critical for detecting bundle arbitrage — a core strategy for our bot analysis.
