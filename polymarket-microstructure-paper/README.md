# Polymarket Microstructure Paper

**Source:** https://github.com/philippdubach/polymarket-microstructure (replication)
**Paper:** https://arxiv.org/abs/2604.24366

Key reference for understanding Polymarket's market microstructure. Contains the replication package for joining order-book feed with on-chain trade data.

Core finding: trade direction from public feed only matches on-chain ground truth ~59% of the time — critical for backtesting accuracy.
