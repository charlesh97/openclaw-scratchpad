# OctoBot-Prediction-Market

**Source:** https://github.com/Drakkar-Software/OctoBot-Prediction-Market
**Type:** Copy trading + arbitrage bot
**Recommendation:** MEDIUM

## What It Does

Open-source trading bot for automated copy trading and arbitrage strategies on crypto prediction markets:

1. **Copy Trading** — Automatically mirrors positions of top traders on Polymarket
2. **Arbitrage** — Detects price discrepancies within Polymarket or across platforms
3. **Simple Interface** — Designed for ease of use, not full customization
4. **OctoBot Ecosystem** — Part of a larger open-source trading bot project

## Why It Matters

A mature, well-supported open-source project in the OctoBot ecosystem. If you're starting from zero and want to experiment with copy trading + basic arbitrage without building from scratch, this is a good starting point.

## Limitations

- Less customizable than building your own system
- Copy trading subject to the skill of the traders you mirror
- Arbitrage component is basic (not the sophisticated 10k-market monitoring found in ImMike's bot)

## Implementability: 3/5

Good for experimentation and learning. Lower priority for production deployment compared to the top YES-rated algorithms found today.
